============
Installation
============


Stable release
--------------

To install spark-xarray, run this command in your terminal:

.. code-block:: console

    $ pip install spark-xarray

This is the preferred method to install spark-xarray, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for spark-xarray can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone https://github.com/andersy005/spark-xarray.git

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL git clone https://github.com/andersy005/spark-xarray.git/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/andersy005/spark-xarray
.. _tarball: https://github.com/andersy005/spark-xarray/tarball/master
